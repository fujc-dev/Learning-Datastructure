#### 操作系统层面线程状态

```text
初始状态、可运行状态、运行状态、休眠状态和终止状态。
```

#### JAVA层面的线程状态

```text
NEW
RUNNABLE
BLOCKED：阻塞，synchronized，竞争没有拿到锁，
WAITING：等待，wait，join
TIMED_WAITING：超时等待，参数里面弄一个超时的值
TERMINATED：终止
```
