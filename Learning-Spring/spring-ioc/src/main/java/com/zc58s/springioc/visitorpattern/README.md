####  结构说明
```text
    结构对象：可以是组合模式，也可以是集合；能够枚举它包含的元素，提供一个接口，允许Visitor访问它的元素。
也就是庄周和甄姬允许外键访问。

    抽象节点：定义一个accept操作，已visitor作为参数，这个Element就是所有英雄的模板。
    
    具体节点：实现Element中的accept方法，调用Visitoe的访问方法，以便完成对一个元素的操作，这个类表示的具体的英雄，
就好比咱玩的韩信，赵云，庄周以及甄姬这些。
    
    抽象访问者：为该对象结构中具体的元素声明一个访问者接口，好像王者荣耀里面不同的玩家。
    
    具体访问者：每个具体的访问者都实现了Visitor中定义的操作，就好比你和梦泪。
```
